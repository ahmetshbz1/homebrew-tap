# Homebrew Tap

This is a Homebrew tap for installing Synapse - a modern clipboard manager for macOS.

## Installation

```bash
brew tap ahmetshbz1/tap
brew install --cask synapse
```

## Available Casks

| Cask | Description |
|------|-------------|
| synapse | Modern clipboard manager for macOS |

## Updating

```bash
brew update
brew upgrade --cask synapse
```

## Uninstalling

```bash
brew uninstall --cask synapse
brew untap ahmetshbz1/tap
```
